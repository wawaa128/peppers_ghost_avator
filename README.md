# 等身大対話型ペッパーズ・ゴースト・システム（MVP）環境構築・起動手順書

本システムは、FastAPI（バックエンド）とThree.js/Web VRM（フロントエンド）を組み合わせ、Gemini 2.5 Flashとedge-ttsによる超低遅延な等身大ペッパーズ・ゴースト対話環境を構築するシステムです。

## 1. 事前準備とディレクトリ構成
任意のPCの作業フォルダ（例として以下を標準とします）に、必要なファイルを配置します。

* **標準の作業フォルダパス**: `C:\Users\staff\workspace\peppers_ghost_avator`

### 配置するファイル一覧
```text
peppers_ghost_avator/
├── server.py               # FastAPI + Gemini + edge-tts (バックエンド)
├── index.html              # Three.js + VRM制御 + UI (フロントエンド)
└── 7151938431140058353.vrm  # 使用する等身大アバターモデル
```
※フロントエンドの3D描画ライブラリ（Three.js / @pixiv/three-vrm）はCDN経由で自動取得するため、Node.js等のインストールは**不要**です。

---

## 2. 【初回のみ】環境構築手順

新規PCで初めてシステムを動かす場合は、以下の手順でPythonの仮想環境（`venv`）を構築し、依存ライブラリをインストールします。

### ① コマンドプロンプトの起動と移動
コマンドプロンプトを開き、作業ディレクトリへ移動します。
```cmd
cd C:\Users\staff\workspace\peppers_ghost_avator
```

### ② Python仮想環境（venv）の作成
フォルダ内に「venv」という名前の独立したPython環境を構築します。
```cmd
python -m venv venv
```
※ `python` コマンドでエラーが出る場合は `py -m venv venv` を試してください。

### ③ 仮想環境の起動（有効化）
作成した仮想環境に入ります。
```cmd
venv\Scripts\activate
```
※成功すると、プロンプトの先頭に `(venv)` と表示されます。

### ④ 依存ライブラリのインストール
仮想環境が有効な状態（先頭に `(venv)` がある状態）で、以下の必須パッケージを一括インストールします。
```cmd
pip install fastapi uvicorn websockets google-generativeai edge-tts
```

## 3. 【2回目以降】システムの起動手順

環境構築が終わっている環境、または日々の運用時はこの手順だけで起動します。

### ① コマンドプロンプトを起動し、ディレクトリへ移動
```cmd
cd C:\Users\staff\workspace\peppers_ghost_avator
```

### ② Python仮想環境の起動
```cmd
venv\Scripts\activate
```

### ③ Gemini APIキーの環境変数セット
利用するGemini APIのキーを一時的に環境変数に登録します（コマンドプロンプトを閉じるまで有効）。
```cmd
set GEMINI_API_KEY=あなたの実際のAPIキー
```

### ④ ローカルサーバー（バックエンド）の起動
```cmd
python server.py
```
ターミナルに `Application startup complete.` および `Uvicorn running on http://0.0.0.0:8000` と出ればバックエンドの起動は成功です。

### ⑤ Webブラウザからフロントエンドにアクセス
1. **Google Chrome** を起動します（Web Speech APIを使用するためChrome必須）。
2. アドレスバーに以下のURLを入力してアクセスします。
   ```text
   http://localhost:8000/static/index.html
   ```
3. 画面中央の下部に**「システム起動準備完了（スペースキーを叩いてください）」**と1秒周期でじわじわ明滅表示されるまで待ちます。
4. **【Space】キー**（または画面内のエメラルド色の起動ボタン）を押します。
   * ※初回のみブラウザからマイクの利用権限を求められるので、必ず**「許可」**をクリックしてください。
5. 画面最上端に**細い赤い帯（マイクONインジケータ）**が出現し、左下に「何か 話しかけてください...」と表示されれば、完全スタンバイ状態です。

## 4. 現場運用・現場調整（チューニング）機能

展示会場やラボ環境の特性に合わせ、コードを1行も書き換えることなく現場でフィッティングを行えます。

* **【Esc】キーによる設定オーバーレイ画面の開閉**
  ページロード直後（スペースキーを押す前）、および会話中のいつでも `Esc` キーで「SYSTEM OPTIONS」画面をポップアップできます。操作中は安全のためマイクが自動的に一時停止します。
* **画面左右反転モード（正像・鏡像切り替え）**
  ハーフミラーへの背面投影ではなく、PC内蔵モニターで直接デバッグ・スライダー操作を行いたい場合は、オプション画面内の「画面左右反転モード」を「反転 OFF」にするだけで、一瞬で文字やUIが正しい向きに戻ります。
* **実機位置調整・ビジュアル色味補正**
  スライダーを動かすだけで、アバターの `X軸/Y軸位置`、`全高Scale`、白飛びを抑えるエグめのトーンフィルター（`Brightness/Contrast/Saturate`）がその場でリアルタイムに変形・反映されます。
* **音声・キャラクターボイス切り替え**
  話速（`Rate`）や声の高さ（`Pitch`）、女性声（Nanami/Aoi）や男性声（Keita/Daichi）の切り替えがコンボボックスから自由に行えます（変更後に話しかけた会話から適用されます）。

## 5. 開発者向けトラブルシューティング

* **アバターが読み込めない（CORSエラー等）**
  * `index.html` ファイルをブラウザに直接ドラッグ＆ドロップ（`file://` スキーム）するとセキュリティ制限でVRMがロードできません。必ず `python server.py` を起動した状態で、`http://localhost:8000/...` 経由でアクセスしてください。
* **音声合成で `Invalid pitch '...%'` というエラーが出る**
  * edge-ttsの仕様上、話速（Rate）は `%` で受け付けますが、ピッチ（Pitch）は `%` だとクラッシュします。必ず **`Hz`** 単位（例: `+30Hz`）で指定してください（現在のUIからは自動的に `Hz` でパッキングされるよう対策済みです）。
* **Geminiへの接続でWebsocketが弾かれる**
  * `server.py` 内のモデル指定が最新の **`gemini-2.5-flash`** になっているか確認してください（旧2.0-flash指定では動作しない場合があります）。
